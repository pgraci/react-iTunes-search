## Installation

1. `git clone git@github.com:LeoAJ/react-iTunes-search.git`
2. `cd react-iTunes-search`
3. `npm install`

## Dev

Run webpack dev server with [react-hot-loader](https://github.com/gaearon/react-hot-loader)

```
npm run dev
```

## Deploy

Use webpack minify plugin

```
npm run deploy
```

## Lint

run eslint

```
npm run lint
```
